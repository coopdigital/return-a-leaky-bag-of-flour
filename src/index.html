Return a leaky flour bag accessibility exercise instructions

'Return a leaky flour bag' is a made-up Co-op journey, filled with accessibility issues. Your task is to find them all!

To help find the issues, you will need to use the WAVE accessibility plugin with either Chrome or Firefox. You'll also need to try to complete the exercise using just your keyboard. You should use this worksheet as you are completing the task.

There are 5 pages in total:

  • Contact us
  • Can I return my flour?
  • Return my flour
  • Check answers
  • Confirmation

Accessibility issues start on the next page. Good luck!
